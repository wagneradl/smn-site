# Templates para Microsoft Forms - SMN Site

## 📋 Visão Geral

Esta pasta contém templates padronizados para facilitar a coleta de conteúdo via Microsoft Forms. Como o Forms tem limitação de 4000 caracteres por campo, criamos templates externos que podem ser baixados, preenchidos e enviados como anexo.

## 📁 Estrutura

```
templates/
├── forms/              # Templates principais
│   ├── case-template.*    # Template para Cases de Sucesso
│   ├── blog-template.*    # Template para Artigos do Blog
│   └── examples/          # Exemplos preenchidos
└── assets/             # Recursos auxiliares
    ├── logo-smn.png       # Logo oficial
    └── instrucoes-*.pdf   # Guias de instrução
```

## 🎯 Como Usar

### Para Cases de Sucesso

1. Baixe `case-template.docx` ou `case-template.txt`
2. Preencha seguindo as instruções
3. Salve com nome: `case-[nome-cliente]-[seu-nome].docx`
4. Envie via Microsoft Forms

### Para Artigos do Blog

1. Baixe `blog-template.docx` ou `blog-template.txt`
2. Preencha seguindo as instruções
3. Salve com nome: `artigo-[titulo-resumido]-[seu-nome].docx`
4. Envie via Microsoft Forms

## 📝 Formatos Disponíveis

- **DOCX**: Para quem prefere Word (formatação visual)
- **TXT**: Para quem prefere texto simples (mais técnico)
- **PDF**: Para visualização e referência
