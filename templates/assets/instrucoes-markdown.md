# 📋 Guia Rápido: Markdown para Microsoft Forms

## 🎯 O que é Markdown?

Markdown é uma linguagem de formatação simples que permite criar textos estruturados usando símbolos especiais. É amplamente usado em documentação técnica e blogs.

## 📝 Sintaxe Básica

### Títulos
```
# Título Principal (H1)
## Título de Seção (H2)  
### Subtítulo (H3)
#### Título Menor (H4)
```

### Formatação de Texto
```
**texto em negrito**
*texto em itálico*
`código inline`
~~texto riscado~~
```

### Listas
```
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2

1. Item numerado 1
2. Item numerado 2
3. Item numerado 3
```

### Links e Imagens
```
[texto do link](https://exemplo.com)
![descrição da imagem](caminho/para/imagem.jpg)
```

### Citações
```
> Esta é uma citação
> Pode ter múltiplas linhas
```

### Código
````
```javascript
function exemplo() {
  return "Bloco de código";
}
```
````

### Tabelas
```
| Coluna 1 | Coluna 2 | Coluna 3 |
|----------|----------|----------|
| Valor A  | Valor B  | Valor C  |
| Valor D  | Valor E  | Valor F  |
```

## ✅ Dicas para o Microsoft Forms

1. **Mantenha a estrutura**: Use ## para títulos principais e ### para subtítulos
2. **Seja consistente**: Mantenha o mesmo padrão de formatação
3. **Teste localmente**: Use um editor Markdown para visualizar antes de enviar
4. **Não use HTML**: Stick com Markdown puro
5. **Quebre linhas**: Use linha em branco entre parágrafos

## 🔧 Editores Recomendados

- **Online**: Dillinger.io, StackEdit
- **Desktop**: Typora, Mark Text
- **VS Code**: Com extensão Markdown Preview
